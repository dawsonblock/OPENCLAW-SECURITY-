# `aetherbot pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
aetherbot pairing list whatsapp
aetherbot pairing approve whatsapp <code> --notify
```
